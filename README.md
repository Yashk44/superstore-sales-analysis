# Superstore Sales and Profitability Analysis

## Overview
This project analyzes retail sales data to evaluate business performance and identify factors affecting profitability. The analysis focuses on loss-making sub-categories, the impact of discounting, and regional performance differences.

## Dataset
- ~9,900 records with multiple features  
- Includes: Category, Sub-Category, Region, Segment, Sales, Profit, Discount, Quantity  
- Data cleaned and prepared for analysis (no missing values detected)

## Tools and Technologies
- Python (Pandas) – Data cleaning and exploratory analysis  
- Power BI – Dashboard development and visualization  
- Excel – Data handling and validation  

## Methodology
- Loaded and explored dataset using Pandas  
- Cleaned data by removing duplicates and standardizing column names  
- Performed aggregation and analysis across category, region, and segment  
- Conducted profitability analysis using sales vs profit comparisons  
- Evaluated impact of discounting on profit  
- Identified profit-driving and loss-making sub-categories  
- Built an interactive dashboard in Power BI  
- Documented findings through report  

## Dashboard
- Displays key KPIs:
  - Total Sales: $2.3M  
  - Total Orders: 9,977  
  - Average Order Value: $230  
- Visualizes:
  - Sales by Category, Region, and Segment  
  - Profit by Sub-Category  
  - Loss by Sub-Category  
- Provides a consolidated view of business performance  

## Results
- High sales do not guarantee profitability (Furniture underperforms)  
- Tables sub-category generates highest loss (~-$17K)  
- Discounts negatively impact profit  
- Consumer segment contributes highest revenue (~$1.16M)  
- West region performs best, Central region underperforms  

## How to Run
1. Clone the repository  
2. Open the Jupyter Notebook to review data cleaning and analysis  
3. Open the Power BI (.pbix) file to explore the dashboard  
4. Refer to the report for detailed explanation  

## Outcome
Delivered a complete analysis workflow from data cleaning to dashboarding, identifying key profitability issues and enabling data-driven decisions on pricing, discount strategy, and product focus.
